# Photochem

`Photochem` contains:

- A one-dimensional photochemical model for estimating the composition of a planet's atmosphere.
- A one-dimensional climate model for estimating the average temperature-pressure profile of a planet's atmosphere.
- A chemical equilibrium solver.

To use `Photochem`, first install it following the [installation instructions](installation.md), then try out either the [Photochemistry] or [Climate] introductory tutorials.
